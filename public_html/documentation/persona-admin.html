Case & Interaction Flows

Case #1

Jackson is checking the site as the admin/moderator and looking for a post that has been flagged by another user.

  • Pre-condition: Pre-condition: As a user and administrator of the site I want to make it easy for both organizations and volunteers to be able to sign up and provide or take opportunities to help their communities.
  • Post-condition: Jackson logged in to the site to check if any posts have been flagged or reported.

Interaction Flow: Admin checks reported post

  • Jackson logs into his account via the login button at the top right of the page.
  • MissionCitizen returns a login form with a blank field for username and password.
  • Jackson enters login details and clicks “Login” button.
  • MissionCitizen site reloads profile page.
  • Inside of profile page the MissionCitizen displays a message box that has postings that have been reported by other volunteers/organizers. Jackson reviews inbox for flagged posts.
  • MissionCitizen returns a posting that has been flagged by an organizer.
  • Jackson clicks on the reported post for details on why it has been reported.
  • MissionCitizen returns a form that was filled out by an organizer.
  • Jackson reads the reason listed [User claiming to be a part of XYZ corporation] Organizers says the post is by someone not known to the organization.
  • Jackson reviews the post by clicking on the link for volunteer opportunity.
  • MissionCitizen returns listing that has been reported.
  • Jackson clicks “Message Post Author” Jackson sends a message to verify association with the organization that was claimed.
  • MissionCitizen returns form with Subject, Body and Send button at bottom of form.
  • Jackson messages the author of post to gather more details. Enters text in form and clicks “Send”.
  • MissionCitizen returns form closes and container pops up with “Message Sent”.
  • Jackson clicks “My Profile” link at top right of the page.
  • MissionCitizen returns Jackson back to original profile page where he started when logged in.
